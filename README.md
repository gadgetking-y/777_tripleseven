# 777 Three Sevens

ポケコン向けアクションパズル「777(Three Sevens)」をWebアプリとして遊べるようにした静的実装です。縦方向にブロックが落ちるWeb向け画面で再現しています。

## 🎮 Webで遊ぶ

ブラウザで直接遊べます（GitHub Pages）:
👉 **[https://gadgetking-y.github.io/777_tripleseven/](https://gadgetking-y.github.io/777_tripleseven/)**

## ローカルでの起動

`index.html` をブラウザで開くか、ローカルサーバーで確認します。

```sh
python3 -m http.server 5173
```

その後、ブラウザで `http://localhost:5173` を開きます。

## 操作

- 左右矢印: 移動
- 下矢印: 1マス落下
- Space または 5: ブロックを時計回りに回転
- Enter: 一気に落下
- P: 一時停止

操作、固定、消去、ゲームオーバー時に控えめな効果音が鳴ります。音源ファイルは使わず、ブラウザ上で短い電子音を生成しています。

## ルール

- 1から6のブロックは、縦または横に3個以上並んだ合計が7になると消えます。
- 7のブロックは、縦または横に3個以上並ぶと消えます。
- 1から6が消えると、隣接する2から6の数字が1減ります。
- 7が消えると、隣接する2から6の数字が半分になります。端数は切り上げます。
- 2個組のブロックは、片方を軸にして時計回りにぐるぐる回転できます。
- 片方だけが引っかかった場合、もう片方は分離して最後まで落ちます。
- 1と7は隣接変化を受けません。
- ハイスコアはブラウザのローカルストレージに保存されます。
